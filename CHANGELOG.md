# 更新日志

## 1.0.2

- 精简扩展详情页资源入口，隐藏仓库、问题、许可证和作者链接等元数据。
- 调整打包忽略规则，避免许可证文件作为资源入口显示。

## 1.0.1

- 优化协同侧边栏响应式布局，统一会话操作按钮和聊天发送区的窄屏换行规则。
- 优化参与者列表展示，人多时改为列表内部滚动，避免继续挤压聊天区域。
- 优化浅色主题下其他参与者消息气泡的背景层次。
- 调整侧边栏活动栏图标，提升重启 VS Code 后的小尺寸可见性。

## 1.0.0

- 首次发布协同编码扩展。
- 支持主机发起会话、参与者通过邀请链接加入会话，以及在新窗口中加入协作。
- 支持工作区文件实时同步、保存状态同步，以及每位参与者独立撤销和重做。
- 支持参与者列表、光标位置跳转、会话内聊天，以及图片/视频/附件消息。
- 支持共享终端和共享本地服务，便于远程联调与协作排查。
